## Integration Notes

This artifact complements the current Multivoice Router Camelot production skeleton:

- `src/runtime/camelot-runtime.ts` provides the runnable spine.
- `src/runtime/command-queue.ts` and `src/runtime/supabase-command-queue.ts` provide queue contracts.
- `src/provenance/*` provides DAG signing, attestation, and ledger events.
- `src/agents/knight-roster-v400.ts` contains the typed Knight roster.
- `src/cartridges/cartridge-registry.ts` contains quest package/mode mounting.
- `docs/architecture/BIO_KINETIC_CARTRIDGE_SYSTEM.md` defines cartridges as quest-level packages, not Knight brains.
- `docs/knowledge/MMAP_MENTAL_MODEL_APPLICATION_PROTOCOL.md` defines mental models as Knight cognition forged by Merlin before GENESIS.

## Golden Rule

The client sees outcomes, interfaces, reports, and approved workflow launchers. The client never sees Camelot kernel prompts, global memory, private orchestration rules, or cross-tenant intelligence.
